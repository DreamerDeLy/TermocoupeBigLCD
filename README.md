# TermocoupeBigLCD
Проект з 2018 року.

![-](https://github.com/DreamerDeLy/TermocoupeBigLCD/blob/master/Pictures/photo_2018-12-24_17-14-44.jpg)

## Features
* Термомостат
* Великий шрифт на LCD
* Настройка спрацювання реле
* Настрйока таймеру включення
* Управління серез енкодер

## Pinout
**Encoder**
* CLK - 2
* DT - 8
* SW - 7

**Terocoupe**
* DO - 4
* CS - 5
* LK - 6

**Relay**
* 13

**LCD (i2c)**
* SDA - A4
* SCL - A5
